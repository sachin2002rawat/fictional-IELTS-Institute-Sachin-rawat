## Overview
This is a modern, responsive home page for a fictional IELTS Institute built using **React JS** and **Tailwind CSS**. The design emphasizes a clean, professional look and focuses on user experience.

## Features
- **Navbar:** Logo + menu links with smooth scroll navigation  
- **Hero Section:** Headline, subtext, call-to-action button, and banner image  
- **Features Section:** 4 cards highlighting key offerings (Speaking Practice, Mock Tests, AI Band Score, Live Doubt Clearing)  
- **Testimonials:** 2–3 student reviews with profile images and feedback  
- **Footer:** Contact info, quick links, and social media icons  

## Design Choices
- **Modern look:** Gradient headings, subtle shadows, and color accents  
- **Responsive:** Works seamlessly on mobile, tablet, and desktop  
- **Interactive:** Hover effects and smooth transitions enhance user engagement  
- **Readable & clear:** Proper spacing, typography, and visual hierarchy  

##Design Choices:
The website features a modern, responsive layout that adapts seamlessly to mobile, tablet, and desktop screens. Clean typography, gradient headings, and subtle shadows create a professional and visually appealing look. Hover effects on links, buttons, and cards enhance interactivity and engagement. Color accents guide the user’s attention and improve readability, while smooth transitions make the interface feel dynamic and polished. The feature and testimonial sections highlight key offerings and student experiences in a clear and organized way. The footer provides essential links and contact information with modern styling, maintaining a cohesive visual theme. Overall, the design prioritizes clarity, usability, accessibility, and an intuitive experience, making it engaging and easy for students to explore IELTS preparation resources.
